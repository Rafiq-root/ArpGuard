# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security issue in **ArpGuard**, please do not open a public issue.

Instead, please email the repository owner directly or create a private pull request. We take security seriously and will respond promptly.
